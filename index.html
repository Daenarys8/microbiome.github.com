Tools and source code

The Github site lists several tools for microbiome analysis, including:

  • The microbiome R package is based on the standard phyloseq data format and includes a comprehensive set of reproducible examples for large-scale microbiome analysis and visualization
  • The HITdb dedicated reference sequence database for improved 16S rRNA based gut microbiome profiling
  • The NMGS neutral models for microbial metagenomics
  • The early warnings toolkit to analyze shifts between alternative stable states in complex (eco)systems
  • The HITChipDB R package: custom tools for HITChip phylogenetic microarray analysis
