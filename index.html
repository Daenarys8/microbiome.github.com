Welcome

The microbiome project collects source code for molecular ecological studies. This emerging code base is provided freely under the FreeBSD open source license.

Tools and source code

We host several R packages suited for microbial ecological studies. See the full list of R packages at the Github site.

The microbiome R package includes various standard routines for analysing microbiomics profiling data.

A collection of practical reproducible examples with source code is available via the microbiome vignette.

Authors and Contributors

The project is jointly developed by researchers from the Molecular Ecology group, Laboratory of Microbiology, Wageningen University, Netherlands, and Department of Veterinary Bioscience, University of Helsinki, Finland. See contact information for details.
