Welcome

The microbiome project collects source code for microbiomics research. The project facilitates in-house code sharing and collaboration between Wageningen University and University of Helsinki. The emerging code base is provided freely also for third parties under the FreeBSD open source license. The tools are in beta development stage. For further information, check the microbiome wiki or contact the admins.

Source code

The project hosts the microbiome R package, which includes standard R routines for preprocessing, analysis and visualization of microbiomics profiling data, in particular for HITChip analysis. For installation instructions and other documentation, see the wiki pages.
