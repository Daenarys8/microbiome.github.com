Welcome

The microbiome project collects source code for microbiome research and facilitates in-house code sharing and collaboration between research groups from Wageningen University and University of Helsinki. The emerging code base is provided freely also for third parties under the FreeBSD open source license. The tools are in beta.

Tools and source code

The microbiome R package includes standard routines for the preprocessing and analysis of microbiomics profiling data. For installation instructions and other documentation, check the wiki.

Authors and Contributors

The project is jointly developed by researchers from the Molecular Ecology group, Laboratory of Microbiology, Wageningen University, Netherlands, and Department of Veterinary Bioscience, University of Helsinki, Finland. The main developers are Leo Lahti and Jarkko Salojärvi.
