Tools and source code

The Github site lists several tools for microbiome analysis.

The microbiome R package includes various standard routines for analysing microbiomics profiling data. Reproducible examples are available via the a href="https://github.com/microbiome/microbiome/blob/master/vignettes/vignette.md">vignette.

Authors and Contributors

The main contributors include researchers from the Molecular Ecology group, Laboratory of Microbiology, Wageningen University, Netherlands, and Department of Veterinary Bioscience, University of Helsinki, Finland.

Don't hesitate to contact us for feedback, support, or feature requests.
